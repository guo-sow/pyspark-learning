# pyspark-learning

## 概要 / 概述

### 日本語

このリポジトリは、PySpark を学習するための個人用リポジトリです。

PySpark の基礎から実践的なデータ処理まで、学習過程で作成したノート、サンプルコード、演習問題をまとめていきます。

主な目的：

- PySpark の基本概念を理解する
- DataFrame / SQL / RDD の使い方を学ぶ
- データ処理の実践力を身につける
- パフォーマンス最適化について学ぶ
- データエンジニアリングの基礎を習得する

---
